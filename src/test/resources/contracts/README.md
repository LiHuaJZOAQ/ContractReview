# 合同审查测试数据

## 目录结构

```
contracts/
├── rental/          # 租房合同（应届毕业生场景）
├── labor/           # 劳动合同（求职者场景）
└── outsourcing/     # 外包协议（自由职业者场景）
```

每个场景包含：
- `normal_*.txt` — 正常合同，条款合理合规
- `problematic_*.txt` — 问题合同，包含多个风险条款

---

## 租房合同（rental）

### 场景：应届毕业生租房

| 文件 | 类型 | 说明 |
|------|------|------|
| `normal_rental_contract.txt` | 正常 | 合理押金、明确费用分担、7个工作日退押金 |
| `problematic_rental_contract.txt` | 问题 | 多个霸王条款 |

#### 问题合同风险点

| 条款 | 问题描述 |
|------|---------|
| 第三条 租金支付 | 年付制，逾期每日加收月租金3%滞纳金 |
| 第四条 押金 | 押金24000元（3个月租金），甲方可单方没收且无需说明理由 |
| 第五条 提前解约 | 乙方提前解约需支付6个月租金违约金，已付剩余租金不退；甲方可随时单方解约仅需提前7日 |
| 第六条 房屋维修 | 日常维修及主体结构维修费用均由乙方承担 |
| 第七条 费用承担 | 一切费用（含供暖费、垃圾清运费等）均由乙方承担 |
| 第八条 使用限制 | 禁止养宠物、禁止打钉粘贴、21:00后禁止播放音乐、每月访客≤3人次 |
| 第九条 检查权 | 甲方可未通知随时进入检查，每周≥1次 |
| 第十条 续租 | 甲方单方决定是否续租及涨租，幅度不受限制 |
| 第十一条 免责 | 因不可抗力、政府行为、物业管理问题导致的损失甲方均不担责 |
| **第十二条 争议解决** | **指定上海市浦东新区人民法院管辖**（乙方在南京，需跨省诉讼） |

## 劳动合同（labor）

### 场景：求职者入职科技公司

| 文件 | 类型 | 说明 |
|------|------|------|
| `normal_labor_contract.txt` | 正常 | 3个月试用期、依法社保、明确加班规定 |
| `problematic_labor_contract.txt` | 问题 | 多个违法条款 |

#### 问题合同风险点

| 条款 | 问题描述 |
|------|---------|
| 第一条 试用期 | 试用期12个月（违反《劳动合同法》第19条：3年合同最长6个月） |
| 第四条 工资 | 试用期工资6000元（低于转正后50%，违反第20条：不低于约定工资80%） |
| 第四条 延迟发薪 | 甲方有权延迟发放工资，最长60日（违反工资支付规定） |
| 第四条 调薪 | 甲方可单方随时调整工资 |
| 第五条 公积金 | 乙方"自愿放弃"住房公积金（违法） |
| 第六条 试用期解除 | 试用期甲方解除无需支付经济补偿（违法） |
| 第七条 违约金 | 合同期内离职需支付10万元违约金（违反第25条：仅培训和竞业可约定违约金） |
| 第七条 退薪 | 非因甲方原因离职需退还已发工资的50%（违法） |
| 第八条 竞业限制 | 竞业期24个月，补偿金仅1000元/月（低于法定标准），违约金20万元 |
| 第八条 保密 | 保密期限为永久（不合理） |
| 第九条 工具设备 | 乙方自备工作设备，甲方不报销（违反劳动保护义务） |
| 第十条 知识产权 | 离职后12个月内成果归甲方（超出合理范围） |
| **第十二条 争议解决** | **甲方所在地有管辖权的人民法院**（保留此表述，对劳动者不利） |

## 外包协议（outsourcing）

### 场景：自由职业者承接软件开发项目

| 文件 | 类型 | 说明 |
|------|------|------|
| `normal_outsourcing_contract.txt` | 正常 | 里程碑付款、合理验收标准、明确知识产权归属 |
| `problematic_outsourcing_contract.txt` | 问题 | 多个不公平条款 |

#### 问题合同风险点

| 条款 | 问题描述 |
|------|---------|
| 第二条 交付延期 | 延迟交付每日扣项目总费用5%，超过7天需退还全部已收款 |
| 第三条 付款 | 甲方可随时无理由拒绝支付尾款，且无需说明原因 |
| 第四条 验收 | 验收标准为甲方主观满意度，可无限次要求返工 |
| 第五条 知识产权 | 离职后独立开发的相似作品归甲方（违法） |
| 第五条 限制 | 禁止在任何平台发布项目相关信息（过度限制） |
| 第六条 竞业 | 竞业期36个月，竞争对手范围由甲方单方界定（含所有互联网公司） |
| 第六条 违约金 | 竞业违约金50万元 |
| 第七条 保密 | 保密义务为永久，违约赔偿不低于100万元 |
| 第八条 工作制 | 全职排他，每日≥10小时，随时响应包括周末节假日（违法） |
| 第九条 责任豁免 | 甲方不对乙方人身伤害担责（违法） |
| 第十条 解约 | 甲方可随时单方解约无需担责，已付款不退；乙方不得解约，擅自解约赔200% |
| **第十一条 争议解决** | **指定广州市天河区人民法院，禁止管辖异议** |

## 使用说明

### 前端测试（单元测试）

```bash
cd web
npm test
```

### 后端测试（JUnit）

```bash
mvn test
```

### 手动测试

将 `problematic_*.txt` 文件内容复制到 PDF 中，通过前端上传并提交审查，验证系统能否正确识别风险条款。

## 风险类型说明

| 风险等级 | 说明 |
|---------|------|
| HIGH | 明显违法或严重损害一方权益的条款 |
| MEDIUM | 存在不公平但不直接违法的条款 |
| LOW | 措辞模糊或约定不明确的条款 |

| 风险类型 | 说明 |
|---------|------|
| 违约金 | 约定过高违约金或违法收取违约金 |
| 免责 | 不合理免除一方责任 |
| 管辖 | 约定对一方明显不利的管辖法院 |
| 其他 | 其他风险条款 |
